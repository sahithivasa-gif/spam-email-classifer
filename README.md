# Spam Email Classifier 📧

## 📌 Overview
The Spam Email Classifier is a machine learning project that classifies emails as spam or not spam. It uses Natural Language Processing (NLP) techniques to analyze text data and identify unwanted or harmful messages. This project demonstrates how ML can be applied to real-world problems like email filtering.

## ⚙️ Tech Stack
- Python  
- Pandas  
- Scikit-learn  
- TF-IDF Vectorizer  

## 🚀 Features
- Classifies emails as Spam or Not Spam  
- Uses text preprocessing techniques  
- Simple and efficient ML model  
- Accepts custom user input  

## 📊 How It Works
The dataset is first cleaned and preprocessed. Text data is converted into numerical format using TF-IDF vectorization. A Naive Bayes model is then trained on the data to classify messages. The trained model predicts whether new input messages are spam or not.

## 📁 Dataset
The model is trained on a dataset containing labeled spam and ham (non-spam) messages.

## ✅ Results
The model achieves good accuracy and performs well on test data.
